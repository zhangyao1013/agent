# 激励增长 · 竞品情报台

面向激励增长业务的竞品动态追踪页面，包含三个 Tab：

- **竞品资讯**：拼多多、快手/极速版、红果短剧、海外 App 等激励增长动向（初始数据来自公开报道检索）
- **竞品评测**：评测报告列表，支持上传（二级页）、业务标签、截图/录屏、文档链接
- **厂商动态**：平台方战略与对激励业务的影响

## 本地运行

```bash
cd agent
npm install
npm run dev
```

浏览器打开 http://localhost:5174

## 说明

- 评测报告与附件保存在浏览器 `localStorage`，适合内部分享原型
- 资讯数据可在 `src/data/seedNews.ts` 中补充你收集的微信文章、研报要点
- 新报告发布后顶部滚动通知可关闭；进入「竞品评测」Tab 后标记为已读
